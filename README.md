# hawiyat-claude-quickstart

> Your AI engineering partner. Build apps, finish freelance work, ship real products — with Claude Code fully connected to Hawiyat Composer.

---

## What is this?

`@hawiyat-team/hawiyat-claude` brings Claude Code into your Hawiyat Composer workspace.

Claude Code is an AI that reads your entire codebase, writes features, fixes bugs, refactors code, runs tests, and builds full apps alongside you. Not a chatbot. Not autocomplete. A real engineering tool that works inside your project and actually gets things done.

This package connects it to Hawiyat Composer so you can start working immediately — no config, no manual setup, one command.

**15,000 DA/month. That's 5x cheaper than Claude Max.**

---

## How to get started

### Step 1 — Get your token

👉 [hawiyat.org/services](https://www.hawiyat.org/services?q=claude+code)

Order through the site. The Hawiyat team will contact you on **WhatsApp**, and once the payment is done you'll receive your token.

> Questions? [wa.me/213559555951](https://wa.me/213559555951)

---

### Step 2 — Make sure Node.js is installed

👉 [nodejs.org](https://nodejs.org) — download the LTS version if you don't have it.

```bash
node -v
# should show v18.x.x or higher
```

---

### Step 3 — Install and connect

```bash
npx @hawiyat-team/hawiyat-claude install --token <your-hawiyat-token>
```

Replace `<your-hawiyat-token>` with the token you received. That's it — Claude Code is now connected to Hawiyat Composer and ready to work.

## What can you do with it?

Everything Claude Code can do. Here are some real examples:

**Build a full app**
```
Build me a REST API with Express and connect it to a PostgreSQL database
```

**Fix a bug**
```
There's an error in my auth middleware, find it and fix it
```

**Write a complete feature**
```
Add user registration and login with JWT tokens
```

**Refactor bad code**
```
Clean up this file, it was written by the previous dev and it's a mess
```

**Write tests**
```
Write tests for all functions in /src/utils
```

**Scaffold a new project**
```
Create a Next.js SaaS starter with Tailwind, auth, and a dashboard
```

Claude reads your project, understands the context, and does the work. You review, approve, and ship.
